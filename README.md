# Qiita CLI

* [📘 How to use](https://github.com/increments/qiita-cli)

## 記事の作成

```shell
$ npx qiita new 記事のファイルのベース名
```

## 記事の投稿・更新

```shell
$ npx qiita publish 記事のファイルのベース名

$ npx qiita publish --all
```

## 記事ファイルを Qiita と同期

```shell
$ npx qiita pull
```

## Qiita Preview

```shell
$ npx qiita preview
```
