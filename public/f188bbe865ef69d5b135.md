# はじめに

今回は`PhpStorm`を導入したときに知っておきたかったプラグインを紹介します

https://www.jetbrains.com/ja-jp/phpstorm/

# UI関連

## Japanese Language Pack / 日本語言語パック

UIを日本語に翻訳してくれます

> 日本語言語パックは IntelliJ IDEA、AppCode、CLion、DataGrip、GoLand、PyCharm、PhpStorm、RubyMine、および WebStorm の UI を日本語にローカライズするプラグインです。

https://plugins.jetbrains.com/plugin/13964-japanese-language-pack------

# 作業関連

## Indent Rainbow
インデントが読みやすくなります
![スクリーンショット 2022-01-23 3.45.38.png](https://qiita-image-store.s3.ap-northeast-1.amazonaws.com/0/555632/9844dcf1-0578-a843-c7e2-5c1ba84ca13b.png)

https://plugins.jetbrains.com/plugin/13308-indent-rainbow

## Rainbow CSV

CSVがカラフルになるので読みやすくなります

https://plugins.jetbrains.com/plugin/12896-rainbow-csv

## deep-assoc-completion

連想配列のキーを自動補完してくれます

https://plugins.jetbrains.com/plugin/9927-deep-assoc-completion

## Laravel Query

Queryを自動補完してくれます

https://plugins.jetbrains.com/plugin/16309-laravel-query

# その他

## Nyan Progress Bar

プログレスバーが猫アイコンになるので幸せな気持ちになります

https://plugins.jetbrains.com/plugin/8575-nyan-progress-bar

## Gitmoji Plus: Commit Button

GitHubのコミットメッセージに絵文字をつけることができます

https://plugins.jetbrains.com/plugin/12383-gitmoji-plus-commit-button
